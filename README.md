# Nandish® — Ultra Premium Graphic Designer Portfolio

A responsive React + Vite portfolio designed around cinematic art direction, premium typography and scroll-triggered motion.

## Highlights

- Asset-aware cinematic intro loader with real image preload progress
- Scroll-directed hero with layered artwork cards
- Dual kinetic marquee lanes
- Sticky manifesto with word-by-word reveal
- Desktop vertical-scroll-to-horizontal featured showcase
- Responsive mobile featured project flow
- Bundled artwork imports to prevent broken image paths
- Skeleton loading, decode handling and image fallbacks
- Animated project filters and premium bento archive
- Fullscreen case-study viewer with keyboard navigation
- Sticky four-stage creative process sequence
- Responsive desktop, tablet and mobile layouts
- Reduced-motion accessibility support

## Run locally

```bash
npm install
npm run dev
```

Open the local Vite URL shown in the terminal.

## Production build

```bash
npm run build
npm run preview
```

The deployable output is generated inside `dist/`.
